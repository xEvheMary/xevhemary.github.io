Descriptions

KopInput adalah aplikasi input data pajak yang dibuat dengan tujuan mempermudah pengguna dalam memasukkan data pajak kedalam template excel yang diberikan. Aplikasi ini juga dapat menyimpan data konsumen dan produk yang dapat digunakan untuk otomatisasi input data pajak.


Cara Menggunakan

  • isikan path file excel yang akan digunakan, tombol disebelah form isian dapat digunakan untuk mencari, atau bahkan membuat file excel baru.
  • Buka tab input, isikan data konsumen dan produk sesuai data masing-masing.
  • Simpan data konsumen dan produk masing-masing dengan menekan tombol "Save".
  • Periksa data yang telah dimasukkan pada tab Display.
  • Export data ke template excel dengan menekan tombol "Save".
